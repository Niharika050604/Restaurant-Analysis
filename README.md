# Restaurant Analysis 🍽️

A data analysis project exploring restaurant data to uncover business insights using Power BI.

## 📁 Files in this Repository
| File | Description |
|------|-------------|
| `restaurants.csv` | Raw dataset containing restaurant records |
| `Practical - 1.pbix` | Power BI dashboard with visualizations |
| `Practical_1_word_doc.docx` | Written analysis and findings |
| `_Questions practical.pdf` | Problem statement / questions answered |

## 📊 Objective
To analyze restaurant data and identify patterns in ratings, locations, cuisines, and pricing to support business decision-making.

## 🔍 Key Areas Explored
- Distribution of restaurants by location and cuisine type
- Relationship between price range and customer ratings
- Top-performing restaurant categories
- Insights to improve customer satisfaction
